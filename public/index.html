Northcoders News API

A RESTful API for accessing and interacting with news articles, topics, users, and comments.

/api /api/topics /api/articles /api/articles/:article_id /api/articles/:article_id/comments POST /articles/:article_id/comments PATCH /articles/:article_id DELETE /comments/:comment_id /api/users

GET /api

Returns all available API endpoints.

GET /api/topics

Serves an array of all topics.

{
  "topics": [
    { "slug": "football", "description": "Footie!" }
  ]
}

GET /api/articles

Responds with an array of all articles.

Query Parameters:

  • sort_by: column name (default: created_at)
  • order: asc or desc (default: desc)
  • topic: e.g. coding 
{
  "articles": [
    {
      "title": "Seafood substitutions are increasing",
      "topic": "cooking",
      "author": "weegembump",
      "created_at": "2018-05-30T15:59:13.341Z",
      "votes": 0,
      "comment_count": 6
    }
  ]
}

GET /api/articles/:article_id

Returns a specific article by ID.

{
  "article": {
    "title": "Living in the shadow of a great man",
    "topic": "mitch",
    "author": "butter_bridge",
    "body": "I find this existence challenging",
    "created_at": "2025-06-05T12:34:56.789Z",
    "votes": 100,
    "article_img_url": "https://images.pexels.com/photos/158651/news-newsletter-newspaper-information-158651.jpeg?w=700&h=700",
    "comment_count": 6
  }
}

GET /api/articles/:article_id/comments

Returns all comments for the specified article.

{
  "comments": [
    {
      "comment_id": 1,
      "votes": 10,
      "created_at": "2025-06-05T12:34:56.789Z",
      "author": "jessjelly",
      "body": "Great article!",
      "article_id": 1
    }
  ]
}

POST /api/articles/:article_id/comments

Adds a new comment to the article.

Request Body:

{
  "username": "string",
  "body": "string"
}

Response:

{
  "comment": {
    "comment_id": 12,
    "body": "This is a comment.",
    "article_id": 1,
    "author": "butter_bridge",
    "votes": 0,
    "created_at": "2025-06-05T12:34:56.789Z"
  }
}

PATCH /api/articles/:article_id

Updates the vote count of a specific article.

Request Body:

{
  "inc_votes": number
}

Response:

{
  "article": {
    "article_id": 1,
    "title": "Running a Node App",
    "body": "This is part two of a series on how to get up and running with Node.js...",
    "votes": 101,
    "topic": "coding",
    "author": "jessjelly",
    "created_at": "2020-11-07T09:00:00.000Z"
  }
}

DELETE /api/comments/:comment_id

Deletes a comment by ID.

{
  "status": 204,
  "body": null
}

GET /api/users

Returns a list of all users.

{
  "users": [
    {
      "username": "rogersop",
      "name": "paul",
      "avatar_url": "https://avatars2.githubusercontent.com/u/24394918?s=400&v=4"
    }
  ]
}
